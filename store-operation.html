🛍️關於我們的精品代購服務 ⌄

感謝您選擇 MURAIN 沐雨精品的代購服務。 我們的商品皆由美國姊姊親自於 美國官方網站 或 Outlet 店鋪 精選購買， 並提供 免費運送，讓您無需擔心額外運費問題。

🇺🇸美國 Outlet 購物須知 ⌄

包裝說明：Outlet 店內商品通常為裸包出售，不附原廠包裝，但請放心，我們會用心為每件商品進行包裝，確保完美送達您的手中。

附贈小禮：每件商品將隨附：

  • 訂製防塵袋
  • 香氛片

若需送禮，請於備註中標註 「送禮需求」，我們將 免費贈送訂製精品紙袋，讓您的心意更完整傳遞。

🔍關於精品的瑕疵說明 ⌄

Outlet 商品為開放展示選購，可能會出現輕微瑕疵，屬正常現象， 包含但不限於：輕微刮痕 / 殘膠 / 小汙點 / 線頭或縫線不完美 / 拉鍊略卡 等。

我們承諾： 所有商品出貨前均經過嚴格人工檢查，確保無明顯重大瑕疵，讓您買得安心、用得放心。

若您對商品完美度有極高要求，建議選擇 專櫃商品， 並理解即使為專櫃商品亦可能存在極輕微細節不完美。

📦瑕疵問題處理 ⌄
  • 反饋時限：如收到商品後發現瑕疵，請確保商品未使用（吊牌未剪、未下水），並於 當日內 提出問題。
  • 處理步驟：請提供清晰開箱影片與照片，客服將協助確認問題並提供解決方案。

重要提醒： 我們對每件商品出貨均錄影存證，建議您收到商品時也 全程錄影， 檢查數量與內容。若未提供開箱影片或照片，恕無法處理後續問題。

🎨色差說明 ⌄

由於每個螢幕顯示器的顏色呈現有所差異，可能導致輕微色差。 基於此，恕無法接受因顏色問題的退換貨申請。 如有顧慮，建議至專櫃實地確認商品後再進行下標。

↩️精品退換貨政策 ⌄
  • Outlet 商品為優惠價格，僅接受 明顯瑕疵超過 1cm 的情況進行退換貨。
  • 不接受退換的情況：皮質紋路 / 壓痕 / 髮絲紋 / 五金輕微刮痕 / 車線不完美 / 殘膠 / 外盒損傷。
  • 若您對商品有較高標準，建議至專櫃購買，以獲得更符合期待的商品及服務體驗。

退貨流程：退貨將提供 7-11 寄貨編號，無需支付額外寄件費。

請妥善包裝商品後，至門市使用 iBon 列印寄件單並完成寄出。
